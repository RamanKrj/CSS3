The box-sizing property

I'm a BOX with width and height 200px, border 10px and padding 10px. The default box-sizing property content-box is in action.So the total width of the box is 200 + 5x2 + 10x2 = 230px
I'm a BOX with box-sizing property set to border-box. The width of the box is unaffected by the padding 10px and border 30px. This is not the case when one sets the box-sizing property to default which is content-box.
